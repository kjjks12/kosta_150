쓰레드!!!!
프로그램 여러개 돌리는거 음악, 그림판 => 멀티 프로그램

멀티 쓰레드=  한 프로그램에서 여러 쓰레드가 동시에 일을 하는거 ex) 음악 , 작업 ,시계

Start() 하면 JVM에서 public void run 을 호출해서 시작해!!! 
니가 run 시작하면 안되!!!




데몬 쓰레드(setDaemon, isDaemon)
메인이 끝나면 알아서 종료되는 쓰레드!!
예를 들어 시계 같은경우 평생 돌려야 되자나!!!


join()
현재 쓰레드 끝날떄까지 다른 쓰레드가 접근 못하게 함!!
일 먼저 끝내도 특정 쓰레드 끝날때까지 죽지 못하게 막아!!


yield()
서로 양보하면서 하는거야!!! => 대기상태가 아니라 준비상태로 가!!


Synchronized( 임계 영역 만들겠다!!)
Synchronized(동기화 블럭) 안에서만 쓸수 있는 메써드..
모니터링 기법 : 이놈이 제대로 lock을 쥐고 있는지 확인해야지!!
wait();// 너 lock 을 너무 많이 가지구 있었어 좀 쉬어!!
notify();//wait 에 의해 잠자고 있는놈들 꺠우는거 => 다시 준비상태로 가는거야!! (실행으로 들어갈수 있는!)
notifyAll();//wait 에 의해 잠자고 있는놈들 꺠우는거
